## Getting Started

### Installation


1. Clone the repo
   ```sh
   git clone https://github.com/your_username_/Project-Name.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Enter your Db detail in `.env file`
   ```JS
    DBURL=Db url 
    NODE_ENV=DEVELPMENT
    EMAIL=EMail
    PASSWORD=password for your email
   ```
4. Run the program by 
```sh
  node index.js
```
